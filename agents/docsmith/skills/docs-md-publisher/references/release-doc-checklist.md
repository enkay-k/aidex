# Release Documentation Checklist

- [ ] Docs describe current behavior and expected next behavior.
- [ ] `docs/README.md` links are valid.
- [ ] Changed files have updated module docs.
- [ ] Changed flows have updated `.mmd` diagram sources.
- [ ] Mermaid diagrams render in `docs/site/`.
- [ ] HTML site regenerated with `--clean`.
- [ ] Generated HTML pages link back to Markdown source.
- [ ] Shell command examples are executable in current repo.
