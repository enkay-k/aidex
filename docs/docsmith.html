What is Docsmith?

Docsmith (formerly codenamed Calliope) is the documentation specialist in the AIDE X team. It ensures that documentation is never an afterthought, but a living artifact that evolves with the code.

Core Capabilities

  • Release Docs: Generates changelogs and release notes automatically at every release point.
  • On-the-Fly Docs: Scaffolds module documentation for new code files instantly.
  • Site Building: Compiles Markdown source into a static HTML documentation site (like this one).

How It Works

Docsmith follows the AIDE X agent structure:

  1. Skill Definition (`SKILL.md`): Defines the agent's capabilities and prompt instructions for the LLM.
  2. Tools (`utils/`): Python scripts that perform the actual work (file scanning, HTML generation).
  3. Templates (`docs/templates/`): Standardized formats for documentation to ensure consistency.

Integration Guide

To use Docsmith in your project:

# 1. Copy the agent toolkit
cp -R agents/docsmith/utils/docs your-repo/utils/

# 2. Run the generator
python3 utils/docs/build_docs_site.py --clean
