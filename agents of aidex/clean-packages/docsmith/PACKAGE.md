# Docsmith Clean Package

This package contains agent contract assets and workspace starter files only.

Included:
- agent contract/prompt/skills/utils from `docsmith/`
- workspace starter files (AGENTS/SOUL/session_starter when available)
- shared files: `agent_template.v0.02.yaml`, `command_playbook.md`

Excluded:
- project runtime memory
- conversation logs
- certificates and release artifacts
- hidden local files (for example `.DS_Store`)
