# Ribhu Session Starter

Use this checklist at the start of each new session.

1. Confirm host/runtime: VS Code, Codex, Claude Code, Gemini CLI, Antigravity, or OpenClaw.
2. Load contract in order:
   - `.aidex/agents/ribhu/agent_contract.v0.02.yaml`
   - `.aidex/workspaces/ribhu/AGENTS.md`
   - `.aidex/workspaces/ribhu/SOUL.md`
3. Confirm current lifecycle phase: `-1→0`, `0→1`, `1→10`, or `10→growth`.
4. Confirm objective type:
   - problem shaping
   - architecture blueprint
   - WBS generation
   - tracker handoff
5. Ask 1–3 focused gap questions if constraints are missing.
6. Output only current stage artifacts; do not move to next stage without explicit approval.
